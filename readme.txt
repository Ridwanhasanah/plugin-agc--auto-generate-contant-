Cara menggunakan plugin
1. gantilah link url sesuai dengan nama yang ingin anda cari di wikipedia
2. ketika mengubah link biarkan link home tetap ada cukup tambahkn www.home/cari sesuatu
3. enter , jika keyword yang di cari tidak ada di wikipedia maka akan menampilkan web 404 not page
4. jika berhasil akan menampilkan halaman berisi sesuatu yang anda cari
5. Pugin ini juga mempunya widget untuk di tampikan cukup drag widget keyword agc
6. Plugin ini juga bisa menyimpan daftar list recomendasi pencarian
7. untuk menampikannya list rekomendasi bukalah menu pada dashboard dan pilih menu AGC
